# Exercice 1- Page web – Ada Lovelace (HTML5 / CSS3)

## Description

_Cet exercice consiste à créer une page web simple en HTML et CSS présentant la vie d’Ada Lovelace._

L’objectif est de pratiquer :

- La structure HTML de base
- La mise en forme avec CSS commande de base
- Accrocher une image, et du style à la page

## Technologies utilisées

- **HTML5**
- **CSS3**

## Structure du projet

|\_\_index.html

|\_\_style.css

|\_\_Ada_Lovelace_portrait.jpg

|\_\_README.md

## Contenu de la page :

La page contient :

- Un titre principal
- Un texte de présentation d’Ada Lovelace
- Une image portrait
- Une liste de ses contributions majeures

## Consignes graphiques

- Couleurs : rose et hotpink
- Police : Comic Sans MS
- Mise en page centrée
- Image arrondie avec bordure couleur: hot pink

## Comment lancer le projet

1. Télécharger tous les fichiers du projet.
2. Ouvrir le fichier index.html dans un navigateur web.

## Auteur·e

Projet réalisé dans le cadre d’un exercice HTML/CSS pour une formation de Dev FUll Stack à Ada Tech School de Nantes. <3
