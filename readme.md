# PROJET Adapage : Personnage choisi : Sailor Moon

## Description

Cet exercice consiste à créer une page web simple en HTML et CSS.

L’objectif est de pratiquer :

- La structure HTML de base
- La mise en forme avec CSS commande de base

La page devra contenir :

[x]Un menu de navigation avec un lien vers chaque section (intro, specificités, frize, temoignage)
[x]Une section de présentation (avec photo et description de la personne)
[x]Une section avec 6 cartes :
[x]nationnalité
[x]metier/activité
[x]signe astro
[x]meilleur qualité
[x]Une frize présentant les moments fort de sa vie
[x]Un témoignage / citation célèbre de cette personnalité
[x]Un footer

## Technologies utilisées

- HTML5
- CSS3

## Structure du projet

MagicalGirlsProjet/
├── README.md
├── index.html
├── style.css
├── sailorMoonImages/

## Contenu de la page :

La page contient :

- Un header avec une barre de navigation
- Un main avec une grille contenantpour structurer les sections:
  - Descritption
  - Image
  - Carte informations
  - Frise chronologique
  - Citation
  - Video
- Un footer avec les contacts et le lien vers GitHub

## Comment lancer le projet

1. Télécharger tous les fichiers du projet.
2. Ouvrir le fichier index.html dans un navigateur web.
